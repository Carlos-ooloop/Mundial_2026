# World Cup FIFA API
## API made with MongoDB, Python, JWT and FastAPI 
### Description:
This is an API when you can auth users and authentified them. When you are already auth you can add players and selections, make gets, puts and deletes. The users has the roles of admin and users.
The players has stats like goals,assits, tackles, interceptions, clean sheets... and you can also make consults like topscorers and top assits.
